# 12V-5V-3.3V-Multi-Rail-Power-Supply-Design

Designed a two-stage, multi-rail DC-DC power supply schematic in Altium Designer. Stage 1 uses a LM2596S-5.0 switching buck regulator with a 33µH inductor and MBRS340 Schottky diode for 12V→5V conversion. Stage 2 cascades an LM1117IMP-3.3 LDO for a clean 3.3V output. Included decoupling/filter capacitors and dedicated header connectors for bench-level testing.
